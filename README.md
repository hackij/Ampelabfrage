# Ampelabfrage

Eine einfache Web-App fuer den Unterricht: Schueler scannen einen QR-Code, waehlen Rot, Gelb oder Gruen und halten ihr Smartphone als direkt sichtbare Rueckmeldung hoch.

## Funktionen

- Drei klar erkennbare Ampelfarben
- Grosse Vollbild-Anzeige fuer das Smartphone
- Kurzer Statussatz zur gewaehlten Farbe
- Uhrzeit der letzten Auswahl
- Auswahl bleibt nach dem Neuladen erhalten

## Nutzung

1. `index.html` im Browser oeffnen oder die Dateien auf einen Webspace/GitHub Pages laden.
2. QR-Code auf die URL der Seite verweisen lassen.
3. Schueler waehlen ihre Farbe direkt auf dem Smartphone.
